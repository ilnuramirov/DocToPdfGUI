# CHANGELOG

Все заметные изменения и исправления версий проекта.

## [1.1.0] - 2026-06-17

### Добавлено

- Повышена надёжность открытия документов: четыре уровня попыток (стандартный, ReadOnly, без обновления связей, перезагрузка Word с отключёнными макросами).
- Защита от зависаний: таймауты и автоматическое завершение зависшего процесса `WINWORD`.
- Автоматическое восстановление Word при критических ошибках.
- Оптимизация пауз и сборки мусора для ускорения пакетной обработки.
- Улучшенное логирование с временными метками.

### Исправлено

- Поведение при открытии некоторых DOC-файлов с кириллическими именами.
- Устранение утечек COM-объектов (ReleaseComObject + FinalReleaseComObject).

## [1.0.0] - 2026-06-10

### Первоначальный релиз

- Базовый GUI на Windows Forms.
- Конвертация DOC/DOCX → PDF через COM Word.
- Логирование и простая обработка ошибок.

Если нужно — добавлю файл `RELEASE_NOTES.md` с более подробным описанием каждого исправления или подготовлю теги GitHub для релиза.
